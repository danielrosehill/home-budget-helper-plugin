# Context Directory

This directory stores household-specific context information that helps AI agents provide personalized budget management.

## Directory Structure

- **household/**: Household profile, members, preferences
- **financial/**: Account details, financial institution information
- **goals/**: Financial goals documentation and progress

## Usage

Context files in this directory supplement the main `context.md` file in the repository root.

Use this directory for:
- Detailed context that would clutter main context.md
- Separate documents for different family members
- Reference materials specific to your household
- Historical context that informs current decisions

## Privacy

Files in this directory may contain sensitive personal information. Ensure sensitive files are listed in .gitignore.
